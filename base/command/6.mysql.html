前言

Mysql 这里作为一个关系型数据库拿来使用

# 1.安装过程

# 1.1 获取镜像

docker pull mysql

# 1.2 获取容器

docker run \
-p 3306:3306 \
-e MYSQL_ROOT_PASSWORD=123456 \
--network dockerbetweennetwork \
-itd --name mysql  mysql
