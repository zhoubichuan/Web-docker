# 六.使用 Mysql

::: tip 前言
Mysql 这里作为一个关系型数据库拿来使用
:::

## 1.安装过程

### 1.1 获取镜像

```bash
docker pull mysql
```

### 1.2 获取容器

```bash
docker run -itd --name mysql-test -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 mysql
```

### 1.3 完成安装

- 关闭防火墙 8090 端口
- 打开页面[http://81.71.127.69:8090](http://81.71.127.69:8090)
